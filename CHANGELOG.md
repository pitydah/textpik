# Changelog

## v0.1 - 2026-06-14

Initial release.

### Features
- Popup action bar on text selection
- 13 built-in actions: copy, paste, open URL, Google, YouTube, Maps,
  ChatGPT, DeepSeek, DuckDuckGo, terminal, print, translate, Ollama
- X11 support via QClipboard.Selection
- Wayland support via wl-paste
- KDE KWin cursor bridge for cursor position
- System tray icon with pause/resume, settings, diagnostics
- Configurable settings dialog
- Visual action editor (add/edit/delete/reorder)
- App blacklist
- Numeric shortcuts 1-9
- Theme presets (Custom, Light, Dark, OLED)
- Fade-in animation
- Sticky popup mode
- Click-outside-to-close (focusOutEvent + X11 polling)
- KWin bridge notifyClickOutside for Wayland
- Languages: Spanish / English
